常用的一些单位：

- vw：视窗宽度的百分比
- vh：视窗高度的百分比
- 百分比单位
    在盒子模型中height、width、padding和margin一般都是相对包含块进行计算的

```
之前想做一个渐变的背景，我将背景都设置成了width:100%,但是高度还是没有铺满整个视图,结果将高度设置成width:100vh就好了
```

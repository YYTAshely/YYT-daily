## 行内元素垂直居中

```html
<style>
    div {
        text-align:center;
        line-height:父元素高度;
        /* vertical-align:middle; */
    }
```

## 绝对定位垂直居中

```html
<style>
.parent {
    position:relative;
}
.child {
    position:absolute;
    top:50%;
    left:50%;
    transform:translateX(-50%) translateY(-50%);
}
</style>
```
